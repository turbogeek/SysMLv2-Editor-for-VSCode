# Changelog

All notable changes to the "sysmlv2-lsp" extension will be documented in this file.

## [1.0.7] - 2026-05-13
### Added
- **SysMLv2 & KerML Language Server:** Robust parsing and structural validation leveraging ANTLR4.
- **Syntax Highlighting:** Professional and highly accurate syntax highlighting for `.sysml` and `.ker` files.
- **Snippets:** Included basic snippets for rapid modeling (`part`, `action`, `req`, `usecase`).
- **Validation Rules:** Integrated MBSE best-practice structural validation.
- **VSCode Configuration:** Added extension settings to toggle validation strictly and configure trace logging.

## [1.0.0] - Initial Release
### Added
- Initial project setup.
- Basic `.sysml` support.
